Java Snake Game 🐍

A classic Snake game implementation in Java using Swing.

Features

  • 🎮 Classic snake gameplay
  • 🏃 Multiple difficulty levels
  • 📊 Score tracking
  • 🎨 Smooth graphics

How to Run

  • Install Java JDK 8 or higher
  • Use any Java IDE or command line tools

Controls

  • Arrow Keys: Move Snake
  • Space Bar: Restart

Project Structure

snake-game/
   src/
       App.java
       SnakeGame.java
       Level.java
